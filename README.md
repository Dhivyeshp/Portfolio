## 🚀 Quick Start

### Prerequisites

```bash
node >= 14.0.0
npm >= 6.0.0
```

### Installation

```bash
# Clone the repository
git clone https://github.com/Dhivyeshp/Portfolio.git

# Navigate to project directory
cd Portfolio

# Install dependencies
npm install

# Run development server
npm run dev
```

The site will be available at `http://localhost:3000` 🎉
